# Few Bears

Kompas do najbliższego piwa 🐻🍺 — aplikacja Flutter, która pokazuje kompasem
najbliższy sklep z alkoholem (dane OpenStreetMap/Overpass API) i pozwala
śledzić w "Spiżarni", które piwa już wypróbowałeś/aś.

## Status gotowości do sklepów

- Bundle ID / applicationId: `com.fewbears.app` (tymczasowy — do zmiany na
  docelowy, gdy zostanie zarejestrowana domena/konto deweloperskie).
- Polityka prywatności: dostępna w aplikacji (`/privacy-policy`), treść
  tymczasowa — przed publikacją zaktualizować adres kontaktowy i opublikować
  tę samą treść pod publicznym URL wymaganym przez App Store Connect / Play
  Console.
- Age-gate 18+: obecny przy pierwszym uruchomieniu.
- Ikony i splash: tymczasowe placeholdery w motywie kompasu, do zamiany na
  finalny branding przed publikacją.
- Podpisywanie release (Android keystore, iOS certyfikaty dystrybucyjne) oraz
  konta deweloperskie Apple/Google: jeszcze nie skonfigurowane.
